# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1**, available in full at:

<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

All contributors — human and agent — are expected to read and follow the linked document.

## Enforcement

Report conduct concerns to the project owner:

**Alain Dormehl** — <alaindormehl@gmail.com>

Reports will be reviewed and investigated promptly and fairly. The project owner is obligated to respect the privacy and security of the reporter of any incident.

## Scope

This Code of Conduct applies within all project spaces — the repository, issues, pull requests, discussions, and any associated communication channels — and also when an individual is officially representing the project in public spaces.
